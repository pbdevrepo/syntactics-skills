# FAQ Question Bank

A library of reusable FAQ questions by page type and industry. Use these as starting points
and customize with the client's actual answers. Each answer should be 2–4 sentences, factual,
and self-contained.

## FAQ Writing Rules (Quick Reference)

1. **Question format:** Write as a human would ask an AI agent — natural language, not keyword-stuffed.
2. **Answer length:** 2–4 sentences. No more, no less.
3. **Self-contained:** The answer must make sense without reading the rest of the page.
4. **Include the company name** in at least the first FAQ answer on each page.
5. **Be specific:** If pricing is available, state it. If ranges apply, give ranges. "Contact us for pricing" is an AI dead end.
6. **Use FAQPage schema:** Every FAQ section should have matching JSON-LD (see schema-snippets.md).
